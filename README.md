## ✨ О проекте

**НейроМастер** - это мощный Telegram бот, объединяющий возможности двух передовых нейросетей:

| 🧠 GPT-4o | 🎨 Leonardo AI |
|-----------|----------------|
| Отвечает на вопросы | Рисует по описанию |
| Помнит диалог | Высокое качество |
| Поддержка контекста | Быстрая генерация |

## 🛠️ Установка

```bash
# Клонируем репозиторий
git clone https://github.com/AlexGame2012/NeiroMater_Bot.git
cd NeiroMater_Bot

# Устанавливаем зависимости
pip install -r requirements.txt
```

### 📝 Создайте файл .env
```env
TELEGRAM_BOT_TOKEN=ваш_токен
GITHUB_TOKEN=ваш_токен
LEONARDO_API_KEY=ваш_ключ
```

### ▶️ Запуск
```bash
python bot.py
```
